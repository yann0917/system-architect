# system_architect

系统架构师备考记录

## 本考试设置的科目

1. 信息系统综合知识，考试时间为150分钟，笔试，选择题；
2. 系统架构设计案例分析，考试时间为90分钟，笔试，问答题；
3. 系统架构设计论文，考试时间为120分钟，笔试，论文题。
